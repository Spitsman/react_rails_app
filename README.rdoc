Создать приложение в связке Rails + React
Отобразить список матчей между несколькими командами. При клике на одну из команд получить
- информацию об играх за текущий период,
- максимальное количество забитых и пропущенных голов в сыгранных матчах,
- команду-конкурента, близкую по количеству очков за турнир,
- команду с максимальным отрывом по очкам от данной.
Кнопка - "любимая команда". При нажатии команда помечается соответствующим флагом (в БД) и при дальнейшем выводе в списке матчей выделяется, например, жирным шрифтом. При повторном нажатии флаг снимается.
